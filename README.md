## Pip Usage

Issue commands into a CLI window

### List Packages Installed

python3 -m pip list

### Install / Update

python3 -m pip install [_package-name_]

python3 -m pip install --upgrade [_package-name_]

or

python3 -m pip install update [_package-name_]
